# Tiger vs Lion Classification

Простая нейросеть для классификации изображений тигров и львов на основе ResNet18.

## Структура проекта

```
.
├── data/
│   ├── raw/              # Исходные изображения (тигры и львы в отдельных папках)
│   ├── train/            # Тренировочный набор (70%)
│   ├── valid/            # Валидационный набор (15%)
│   └── test/             # Тестовый набор (15%)
├── src/
│   ├── dataset.py        # Загрузка и предобработка данных
│   ├── train.py          # Обучение модели
│   ├── prediction.py     # Предсказание для новых изображений
│   └── splitDataset.py   # Разделение датасета на train/valid/test
├── requirements.txt      # Зависимости
└── main.py              # Главный скрипт для полного пайплайна
```

## Требования

- Python 3.8+
- PyTorch
- TorchVision

## Установка

```bash
pip install -r requirements.txt
```

## Подготовка данных

1. Положите изображения тигров в папку `data/raw/tiger/`
2. Положите изображения львов в папку `data/raw/lion/`

Затем разделите датасет:

```bash
python src/splitDataset.py
```

Это создаст папки `train/`, `valid/` и `test/` с автоматическим разделением (70/15/15).

## Обучение модели

```bash
python src/train.py
```

Или через главный скрипт:

```bash
python main.py
```

## Предсказание

```bash
python src/prediction.py --image path/to/image.jpg
```

Опциональные параметры:
- `--weights` - путь к весам модели (по умолчанию `./artifacts/best_model.pt`)
- `--data-dir` - путь к датасету (по умолчанию `./data`)

## Модель

- **Архитектура**: ResNet18 с претренированными весами (ImageNet)
- **Transfer Learning**: Заморожены параметры backbone, обновляется только финальный слой
- **Оптимизатор**: Adam (lr=1e-3)
- **Функция потерь**: CrossEntropyLoss
- **Батч**: 16 изображений
- **Эпохи**: 10

## Результаты

Лучшая модель сохраняется автоматически в `artifacts/best_model.pt` по результатам валидационной точности.
